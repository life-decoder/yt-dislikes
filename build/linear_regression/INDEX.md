## 📈 Understanding the Outputs

### Metrics CSV (linear_regression_metrics.csv)
```csv
set,rmse_log,mae_log,r2_log,rmse_raw,mae_raw,r2_raw,mape_raw
training,1.314,1.033,0.348,...
validation,1.312,1.025,0.333,...
```
**Key metrics:**
- `r2_log`: R² on log scale (primary metric) ⭐
- `mae_log`: Mean Absolute Error on log scale
- `r2_raw`: R² on raw scale (can be negative)

### Coefficients CSV (linear_regression_coefficients.csv)
```csv
feature,coefficient,abs_coefficient
comment_sample_size,0.9325,0.9325
likes,0.7930,0.7930
...
```
**Interpretation:**
- Positive coefficient → increases dislikes
- Negative coefficient → decreases dislikes
- Magnitude → strength of effect (after scaling)

### Predictions CSV (linear_regression_predictions.csv)
```csv
set,actual_log,predicted_log,actual_raw,predicted_raw
train,6.5,6.3,665.14,544.23
validation,7.2,7.1,1339.43,1202.56
...
```
**Use for:**
- Error analysis
- Custom visualizations
- Debugging predictions

## 📊 Visualization Guide

### 1. Performance Analysis (6 panels)
**File:** `linear_regression_performance_analysis.png`

**Panels:**
1. **Feature Coefficients** - Bar chart (green=positive, red=negative)
2. **Predictions vs Actual (Log)** - Validation scatter plot
3. **Residuals Distribution** - Error histogram
4. **Predictions vs Actual (Raw)** - Raw scale scatter
5. **Absolute Error Distribution** - Error magnitude histogram
6. **Performance Comparison** - Train vs Validation bars

**What to look for:**
- Points close to red line = good predictions
- Normal residual distribution = assumptions met
- Similar train/val bars = no overfitting

### 2. Raw Scale Predictions (2 panels)
**File:** `linear_regression_raw_scale_predictions.png`

**Panels:**
1. **Training Set** - Raw scale scatter
2. **Validation Set** - Raw scale scatter

**What to look for:**
- Scatter around diagonal = good fit
- Spread increases with magnitude = heteroscedasticity

### 3. Detailed Analysis (6 panels)
**File:** `linear_regression_detailed_analysis.png`

**Panels:**
1. **Residual Plot** - Check homoscedasticity
2. **Q-Q Plot** - Check normality
3. **Error by Video Size** - Performance across sizes
4. **Cumulative Error Distribution** - Percentile analysis
5. **Top 10 Coefficients** - Most important features
6. **Predictions with Error Coloring** - Error density

**What to look for:**
- Flat residual plot = good (not here due to outliers)
- Q-Q points on line = normally distributed
- Consistent error across sizes = robust model

## 🎯 Key Findings Summary

### Performance
- **Validation R² (log): 0.3327** ⚠️ Low (vs XGBoost: 0.8369)
- **Median Error: 556 dislikes** ✅ Reasonable
- **69.4% within ±1K** ✅ Good precision

### Feature Importance
1. comment_sample_size (0.93)
2. likes (0.79)
3. avg_compound (0.66)
4. no_comments (0.65)
5. avg_pos (0.43)

### Overfitting
- **1.49% difference** ✅ Excellent generalization
- Better than XGBoost (5.76%) but much lower R²

### Recommendation
- **Use XGBoost** for this task (2.5x better R²)
- Linear Regression serves as good baseline
- Confirms value of complex models
