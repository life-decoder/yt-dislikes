# Ridge Regression Model

## Model Overview
- **Algorithm**: Ridge Regression with L2 Regularization
- **Target Variable**: `log_dislikes`
- **Best Alpha**: 100
- **Validation R²**: 79.2%
- **Test R²**: 73.4%

## Key Features
- Temporal validation strategy (75-10-15 chronological split)
- Handles multicollinearity through L2 regularization
- Interpretable coefficients for feature importance analysis
- Missing value handling with SimpleImputer
